## Description
FreshFold Laundry is a web-based platform that allows customers to book laundry pickup and delivery services online. It provides live order summaries, automated email/SMS notifications, and backend order management.

---

## Features
- Display item prices clearly.
- Add items to order with quantity selection.
- Live order summary with subtotal, pickup charge, and total.
- Customer details form (name, phone, email, address).
- Automated email and SMS notifications via Nodemailer and Twilio.
- Orders saved locally and on the backend (orders.json).
- Responsive design for desktop and mobile.

---

## Technologies Used
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express.js
- Email: Nodemailer / SendGrid
- SMS: Twilio
- Deployment: Render / Railway / Heroku
- Storage: orders.json
